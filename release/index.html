О нас

Компания “Сфера” была основана в 2010 году для организации эффективного и рационального сотрудничества Высших учебных заведений (Санкт-Петербургский Электротехнический университет, Санкт-Петербургский политехнический университет Петра Великого) и центра разработки ПО “Моторола Солюшнз” в Санкт-Петербурге.

Миссия компании

Вовлечение талантливых молодых специалистов в реализацию проектов по разработке программного обеспечения в области беспроводных коммуникаций.

Нами были успешно разработаны и внедрены

  • Технологии, процессы и алгоритмы тестирования
  • Программный модем для встраиваемых систем
  • Проект "Multi View" (видео стена)
  • iTVphone Адаптер для объединения VoIP телефонии и цифрового телевидения
  • Мобильные приложения (Android, iOS)
  • Платформы сбора и обработки больших данных (в пакетном режиме, в реальном времени), накапливаемых в системе Р25/APCO
  • Прототипы, работающие в беспроводных mesh-сетях
  • Версия MVP, поддерживающая WiFi модуль для "Интернета вещей"
  • Приложения для потоковой передачи звука
  • Алгоритм маршрутизации данных WMN для кампусных, автомобильных, домашних аудио беспроводных сетей
  • Mesh - технология для передачи данных между транспортными средствами в режиме реального времени

Текущие проекты

  • Разработка ПО mesh для транспортно-логистического сектора
  • Mesh технологии для служб экстренного реагирования
